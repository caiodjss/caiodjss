# 🧩 Projetos em Destaque

## 🔹 Compra Mensal — Backend API

API REST desenvolvida para gerenciamento de compras mensais, com foco em autenticação, organização de dados e segurança de acesso.

Além do desenvolvimento da aplicação, o projeto foi utilizado para aprofundar conhecimentos em fluxos de autenticação, análise de requisições HTTP, tratamento de erros e validação de integrações.

### Principais funcionalidades

* Login com Google utilizando OAuth 2.0
* Autenticação baseada em JWT
* CRUD de produtos e categorias
* Rotas protegidas
* Validação de dados de entrada
* Tratamento centralizado de erros
* Integração com PostgreSQL através do Prisma ORM
* Testes unitários com Vitest
* Docker para ambiente de banco de dados
* Padronização de commits com Husky e Conventional Commits

### Tecnologias

Node.js • TypeScript • Express • PostgreSQL • Prisma ORM • JWT • OAuth 2.0 • Docker • Vitest
